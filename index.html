Download and Install It's quite easy

Download and run the installer.

System Requirements

  1. Windows XP Service Pack 2 or better
  2. a video card that supports, at minimum, Shader Model 1.1.
  3. A mouse (or Wiimote and Bluetooth stack)

Using Wiimote?

Install the game as usual. Then follow this article on how to pair your Wiimote with your PC. The game will automatically detect your Wiimote. Hopefully.

Developers

Developers, you can clone the project with Git by running:

$ git clone git://github.com/rofflebuster/Ensembler
Open the solution in Visual Studio and click Build. You'll need XNA Framework 4.
